# TypeScript 濃度・希釈計算ツール

TypeScriptの学習用に作成した、ブラウザで動く小さな計算アプリです。

## 機能

- 質量パーセント濃度の計算
- 希釈時に追加する水の量の計算
- 不正な入力に対するバリデーション
- Vitestによる計算ロジックの単体テスト
- レスポンシブ対応

## 技術構成

- TypeScript
- Vite
- Vitest
- HTML / CSS

## 実行方法

```bash
npm install
npm run dev
```

## テスト

```bash
npm test
```

## ビルド

```bash
npm run build
```
